Positional Number Systems

Digital Data

Data in the computer memory is represented using units that can each be in one of two states (0 and 1).

Data is represented digitally, using binary numbers

Kinds of data:

  • Number : represented in binary
  • Text : each character is mapped to a number
  • Images : matrix of pixels' colors. Each colors represented as (R, G, B) levels (0 - 255)
  • Video : sequence of image
  • Audio : sampled voltage levels

Hello World

In computer science we design algorithms to solve computational problems. We then implement them in a formal programming language which would be executed by a computer.

When stating a computational problem:

  • Define set of legal inputs
  • Define the required output, for each input

Example:

The Sorting Problem

input: A sequence of numbers <a1, a2, ..., an>

output: A reordering <b1, b2, ..., bn> of the input sequence, such that: b1 ≤ b2 ≤ ... ≤ bn

Algorithm

Algorithm for sorting n=2 numbers:

1. print "Please enter two numbers" 2. number1 ← (read a number from the user) 3. number2 ← (read a number from the user) 4. if(number1 < number2): 4.1. print number1 4.2. print number2 else: 4.1. print number2 4.2. print number1

Process of Executing a Program

In computer science we design algorithms to solve computational problems. We then implement them in a formal programming language which would be executed by a computer.

To build Process

High-Level Language: Python 1. print("what is your name?") 2. user_name = input() 3. print("Hi", user_name, ", welcome to python") ↓ translation ↓ Machine Language 01101011 10100010 10011100 : 01001001

Kinds of translations:

  • Compilation

    Complied languages are converted directly into machine code that processor can execute. As a result, they tend to be faster and more efficient to execute than interpreted languages. They also give the developer more control over hardware aspects, like memory management and CPU usage.

    Examples of pure compiled languages are C, C++, Erlang, Haskell, Rust, and Go.

  • Interpretation

    Interpreters run through a program line by line and execute each command. Here, if the author decides he wants to use a different kind of olive oil, he could scratch the old one out and add the new one. Your translator friends can then convey that change to you as it happens.

    Examples of common interpreted languages are PHP, Ruby, Python, and JavaScript.

Addiing Numbers

Problem: write a program that reads from the user two numbers, and prints their sum.

Example

Please enter two numbers in two separate lines:

7
5
7 + 5 = 12 >>> print('Please enter two numbers, in two separate lines:') >>> num1 = input() >>> num2 = input() >>> sum = num1 + num2 >>> print(sum) >>> print(type(sum))
Run: Please enter two numbers, in two separate lines: >? 7 >? 5 75 <class 'str'>

convert default str to int:

>>> print('Please enter two numbers, in two separate lines:') >>> num1_str = input() >>> num1 = int(num1_str) >>> num2_str = input() >>> num2 = int(num2_str) >>> sum = num1 + num2 >>> print(sum) >>> print(type(sum))
Run: Please enter two numbers, in two separate lines: >? 7 >? 5 12 <class 'int'>

shorten code:

>>> print('Please enter two numbers, in two separate lines:') >>> num1 = int(input()) >>> num2 - int(input()) >>> sum = num1 + num2 >>> print(sum) >>> print(type(sum))
Run: Please enter two numbers, in two separate lines: >? 7 >? 5 12 <class 'int'>

print both problem and solution:

>>> output_sentence = num1 + ' + ' + num2 + ' = ' + sum >>> print(output_sentence)
Run: output_sentence = num1 + ' + ' + num2 + ' = ' + sum TypeError: unsupported operand type(s) for +: 'int' and 'str'

adding then converting from 'int' to 'str' for concatenating:

>>> output_sentence = str(num1) + ' + ' + str(num2) + ' = ' + str(sum) >>> print(output_sentence)
Run: Please enter two numbers, in two separate lines: >? 7 >? 5 7 + 5 = 12

using extended version of print; comma for separating any Python data types with a space:

>>> print('abc', 'def', 'ghi') abc def ghi >>> print('abc', 4, 'def', 5.7) abc 4 def 5.7

using extended version of print; comma without converting:

>>> print('Please enter two numbers, in two separate lines:') >>> num1 = int(input()) >>> num2 - int(input()) >>> sum = num1 + num2 >>> print(num1, '+', num2 '=', sum)
Run: Please enter two numbers, in two separate lines: >? 7 >? 5 7 + 5 = 12
